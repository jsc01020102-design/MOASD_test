[비디오 업로드 안내 / Video Upload Instructions]

안녕하세요! 메인 화면의 백그라운드에 첨부하신 (주)MOASD 기업 영상을 재생하기 위해서는 아래 단계를 따라 비디오 파일을 업로드해주셔야 합니다.

대화창에 업로드하신 영상은 AI 모델(Gemini)에 전달되어 분석을 돕지만, 현재 실행 중인 웹 서버의 로컬 저장 공간에 자동으로 저장되지 않으므로 브라우저에서 직접 재생할 수 없습니다.

[적용 방법]
1. 왼쪽 파일 탐색기(File Explorer)에서 이 'public' 폴더를 찾습니다.
2. PC에 있는 비디오 파일의 이름을 'video.mp4' 또는 'hero_video.mp4'로 변경합니다.
3. 준비한 파일을 마우스로 드래그하여 이 'public' 폴더 위치에 놓아 업로드(Drag & Drop)합니다.
4. 파일 탐색기에 'public/video.mp4' 파일이 생성되면 완료됩니다!
5. 브라우저 새로고침 시 즉시 동영상이 메인 화면 전체에 백그라운드로 자동 재생됩니다.

---

[English Guides]
1. Rename your local video file to 'video.mp4' or 'hero_video.mp4'.
2. Drag and drop it into this 'public/' folder inside the code editor's file explorer.
3. Once the file 'public/video.mp4' appears, refresh the website preview to see it running!
